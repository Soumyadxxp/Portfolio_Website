# 🖥️ SoumyaDeep OS

> A Retro-Futuristic Portfolio Operating System built with HTML, CSS, and JavaScript.

SoumyaDeep OS is an immersive web-based portfolio designed to simulate a complete desktop operating system experience inside the browser. Instead of presenting information through a traditional portfolio website, it transforms professional details, projects, certifications, and skills into an interactive operating system environment.

The project combines modern web technologies with retro-inspired visuals to create a unique and engaging user experience for recruiters, employers, and visitors.

---

## 🚀 Overview

SoumyaDeep OS is a fully client-side application developed using HTML5, CSS3, and JavaScript. The application features a realistic boot sequence, desktop environment, window management system, taskbar, start menu, and an interactive terminal that allows users to explore portfolio content through commands.

The goal of this project is to demonstrate creativity, UI/UX design skills, frontend development expertise, and software engineering concepts through an innovative portfolio experience.
